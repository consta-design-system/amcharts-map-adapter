# AmChartsMapAdapter | Дизайн-система Consta

Компонент для отображения чисел — с заголовком, единицами измерения, иконками и другими настройками. Входит в [дизайн-систему Consta](http://uikit.consta.design/).

[Посмотреть на GitHub](https://github.com/consta-design-system/amcharts-map-adapter)

[Посмотреть документацию](?path=/docs/components-amcharts-map-adapter--interactive)

#### Установите пакет и зависимости

`yarn add @consta/uikit @consta/amcharts-map-adapter` или `npm install @consta/uikit @consta/amcharts-map-adapter`

#### Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/amcharts-map-adapter` были вложены в компонент `Theme`.

[Как подключить тему](#http://uikit.consta.design/?path=/docs/thematization-how-to-connect-theme--page)
